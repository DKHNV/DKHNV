# DKHNV

Инженер пусконаладочных работ, системный администратор и специалист по сетевой инфраструктуре.

Профиль посвящён практическим решениям в области слаботочных систем, сетевой инфраструктуры, автоматизации и систем видеонаблюдения.  
Основная часть репозиториев связана с подготовкой и сопровождением сетевых данных сервисов для последующего применения в правилах маршрутизации, фильтрации и сетевой политике.

## О профиле

Этот профиль представляет собой инженерную визитку и подборку рабочих репозиториев.  
Публикуемые материалы отражают практический подход к решению прикладных сетевых задач и подготовке данных для использования в реальной инфраструктуре.

Помимо сетевых решений, занимаюсь пусконаладкой и сопровождением слаботочных систем, систем автоматизации и видеонаблюдения.  
Работаю с решениями MikroTik, Keenetic, HDL, KNX, iRidi, Linux, VPN, а также с системами видеонаблюдения на базе Hikvision, Trassir, Dahua, Интеллект и других профильных платформ.

## Основные направления

- слаботочные системы
- сетевое администрирование
- маршрутизация трафика
- VPN-решения
- автоматизация технических процессов
- системы видеонаблюдения и смежная инфраструктура

## Мои проекты

Репозитории в данном профиле сгруппированы по отдельным сервисам и имеют типовую структуру:

- `*_RouterOS` — репозиторий со скриптом для роутеров MikroTik, автоматически обновляющим address list соответствующего сервиса
- `*_CIDR` — репозиторий со списком IP-адресов и CIDR-диапазонов сервиса
- `*_DNS` — репозиторий со списком доменных имён сервиса

## Примечание

Все материалы в данном профиле публикуются исключительно в образовательных, исследовательских и административных целях.
