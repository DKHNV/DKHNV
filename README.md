# DKHNV

Инженер пусконаладочных работ и системный администратор.

Профиль посвящен практическим решениям в области слаботочных систем, сетевой инфраструктуры, автоматизации и систем видеонаблюдения.  
Основная часть репозиториев связана с подготовкой и сопровождением CIDR-диапазонов сервисов для последующего применения в правилах маршрутизации, фильтрации и сетевой политике.

## О профиле

Этот профиль представляет собой инженерную визитку и подборку рабочих репозиториев.  
Публикуемые материалы отражают практический подход к решению прикладных сетевых задач и подготовке данных для использования в реальной инфраструктуре.

Помимо сетевых решений, занимаюсь пусконаладкой и сопровождением слаботочных систем, систем автоматизации и видеонаблюдения.  
Работаю с решениями MikroTik, Keenetic, HDL, KNX, iRidi, Linux, VPN, а также с системами видеонаблюдения на базе Hikvision, Trassir, Dahua, Интеллект и других профильных платформ.

## Основные направления

- слаботочные системы
- сетевое администрирование
- маршрутизация трафика
- VPN-решения
- автоматизация технических процессов
- системы видеонаблюдения и смежная инфраструктура

## Выбранные проекты

В данном разделе собраны репозитории с CIDR-диапазонами различных сервисов, подготовленными для последующего применения в правилах маршрутизации, фильтрации и сетевой политике.  
Все проекты имеют единую практическую задачу и отличаются только конкретным сервисом, для которого формируются и поддерживаются соответствующие диапазоны.

- [CIDR Telegram](https://github.com/DKHNV/Telegram-AddressList)
- [CIDR YouTube](https://github.com/DKHNV/YouTube-AddressList)
- [CIDR Discord](https://github.com/DKHNV/Discord-AddressList)

## Примечание

Материалы профиля опубликованы исключительно в образовательных, исследовательских и административных целях.  
Они предназначены для изучения принципов сетевой обработки трафика, а также для задач по ограничению, контролю и управлению доступом к отдельным сервисам в рабочей или домашней сети.
